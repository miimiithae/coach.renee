# Renee Makimaa / Wayfinder Coaching Landing Page — Version 3

GitHub Pages ready.

## Upload instructions
1. Unzip this folder.
2. Open your GitHub repository: `coach.renee`.
3. Delete/replace the old `index.html`, `styles.css`, `script.js`, and `assets` folder.
4. Upload everything from this folder into the root of the repository.
5. Commit changes.
6. Go to Settings → Pages and make sure GitHub Pages deploys from `main` branch and `/root`.
7. Open: `https://miimiithae.github.io/coach.renee/`
