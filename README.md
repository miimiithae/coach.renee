# Renee Makimaa / Wayfinder Coaching — V4

GitHub Pages ready. This version removes the awkward repeated flyer image and rebuilds the content as clean, responsive HTML.

## Upload
1. Unzip this folder.
2. Upload `index.html`, `styles.css`, `script.js`, `README.md`, and the full `assets` folder to the root of your GitHub repo.
3. Commit changes.
4. In GitHub: Settings → Pages → Deploy from branch → `main` → `/root`.
5. Open: `https://miimiithae.github.io/coach.renee/`
