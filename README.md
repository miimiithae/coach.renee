# Renee Makimaa / Wayfinder Coaching Landing Page

GitHub Pages ready.

## Upload instructions
1. Unzip this folder.
2. Open your GitHub repository.
3. Delete old files if replacing an existing landing page.
4. Upload `index.html`, `styles.css`, `script.js`, and the full `assets` folder.
5. Commit changes.
6. Go to Settings → Pages and make sure GitHub Pages is set to deploy from the `main` branch and root folder.
